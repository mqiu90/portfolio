# My Portfolio

[Link to Website](https://my-portfolio-indol-alpha-22.vercel.app)

## About

Single-page portfolio website designed to show my journey as a Computer Science student at Cornell University. Key features include mobile-first responsive layout, smooth scroll navigation, animated floating elements, and direct contact links. 

## Tech Stack
- React (with TypeScript)
- Vite
- Lucide React
- Custom CSS
- Vercel 

## Getting Started
Follow these instructions to run the project on your local machine:

1. Clone this repository: git clone https://github.com/mqiu90/portfolio.git
2. Move into the project folder: cd portfolio
3. Install all dependencies: npm install
4. Launch the dev server: npm run dev
5. Navigate to http://localhost:5173 in your browser
   
## Build for Production 
Create an optimized build: 
```bash
npm run build
npm run preview
```
